Single color gradient usages

Apply over an image with text in order to make it more readable

If the text is with light color over also light-colored background image, then single gradient can be added as some kind of filter in order to darken the image.

So add a "new" linear gradient with semi-transparent black color over the image.

Like a gradient colored border

2 gradients added. One with padding-box, that acts as the single-color background. And one with the multi-color gradients that acts as the border
